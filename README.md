Automated Video Subtitling: Goal
================================
Video without subtitles ----> Extract audio from video ----> Signal Processing ----> Speech-to-text ----> English subtitles (.srt) file ----> Translation ----> Hindi .srt file ----> Play video with subtitles

===================================
